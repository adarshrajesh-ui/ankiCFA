Critique (structured senior-designer heuristic — GPT-4o vision UNAVAILABLE, no
OPENAI_API_KEY; same honesty caveat as every other pass):

  D6-1 MAJOR — No brand identity. The dialog was a bare vertical stack of three
    checkboxes + one dense grey paragraph + Save/Cancel. Every other CFA surface
    (Home, Readiness, mobile) carries a brand eyebrow + title; this one had none,
    so it read as a generic Anki add-on sheet, not a premium CFA product.
    FIXED — added a CFA brand heading (eyebrow "ankiCFA · AI" + serif title
    "AI features") via cfa_style.page_heading, so it reads as one product.

  D6-2 MAJOR — Master/feature relationship invisible. The three checkboxes were a
    flat list of equal weight; the master switch (which gates the two below) looked
    like a peer. On master-off the two greyed out, but nothing conveyed the
    parent/child grouping.
    FIXED — the master switch now stands alone with a subtitle; the two per-feature
    switches sit in one indented container under a quiet "PER FEATURE" section
    divider, and the whole group greys out together when the master is off
    (self._features.setEnabled(on)).

  D6-3 MAJOR — No key-status transparency. The entire toggle contract hinges on
    "AND an OpenAI API key is configured", but the dialog never said whether one
    IS — so a user could not tell if the switches would actually reach the model or
    silently fall back. Buried inside a 4-line prose paragraph.
    FIXED — a live status line states the real behaviour up front:
      key present → green (pass) "OpenAI API key detected — AI runs for the
                    switches you enable."
      no key      → orange (warn) "No OpenAI API key set — every feature runs its
                    offline fallback."
    Reads cfa.ai.llm_client.key_present() (the exact gate the AI modules use); the
    key value is never shown or logged.

  D6-4 MINOR — Cramped spacing / dense note. FIXED — consistent spacing scale
    (contentsMargins 20/18, addSpacing between groups), a hairline divider before
    the footnote, and the prose collapsed to a single quiet caption line.
